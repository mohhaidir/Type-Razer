# Lomba Ketik ⌨️

Ini adalah bagian _client_ dari aplikasi.  

## Features Used

  1. Vue.js
  2. Vuex
  3. Vue Router

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Deploy to _firebase_
```
npm run deploy
```
